PINP Blog 开发者手册
----------------------------

---

&nbsp;

拼文（PINP）是一种基于 HTML5 网页的文档系统，它近似于 PPT 软件（微软的 PowerPoint 演示文稿），PINP 是 "PINP Is Not PPT" 的缩写形式，PINP 不是 PPT，但胜似 PPT。

拼文具有两大功能，其一是写文档，等效于 "WORD + PPT"，其二是前端编程，等效于 SPA (Single Page Application) 程序开发。对于前者，请自行学习拼文的 [Online help 文档](https://rewgt.github.io/blogs/)，本手册专注于如做用 PINP 做前端编程。
